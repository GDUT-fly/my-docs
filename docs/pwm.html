三、代码​

  • 根据电机号设置端口、端口号、定时器：motorSetConfig

    函数原型如下：

// 根据电机号设置端口、端口号、定时器
static void motorSetConfig( uint8_t motorId,
                            GPIO_TypeDef *p_dirPort,
                            uint16_t *p_dirPin,
                            TIM_TypeDef *p_tim
                          );
参数说明
motorId电机号 （MOTOR1：一号电机）（MOTOR2：二号电机）
p_dirPortGPIO端口
p_dirPinGPIO端口号
p_tim定时器号
返回值无
  • 设置电机方向：motorSetDirection

​ 函数原型如下：

// 设置电机方向
static void motorSetDirection( GPIO_TypeDef *p_dirPort, 
                               uint16_t dirPin, 
                               Direction direction
                             );
参数说明
p_dirPortGPIO端口
dirPinGPIO端口号
direction旋转方向 （STOP：停止）（FORWARD：往前） （BACKWARD：往后）
返回值无
  • 设置电机方向：motorSetFrequency

​ 函数原型如下：

// 设置频率
static void motorSetFrequency( TIM_TypeDef *p_tim, 
                               uint16_t frequency
                             );
参数说明
p_tim定时器号
frequency频率
返回值无

  • 电机控制：motorControl

    一号电机使用TIM1、二号电机使用TIM3

    软件PWM实现，通过修改定时器频率从而修改定时器中断的频率，从而修改输出电平的频率。

​ 函数原型如下：

// 电机控制
void motorControl( uint8_t motorId, 
                   Direction direction, 
                   uint8_t frequency);
参数说明
motorId电机号 （MOTOR1：一号电机）（MOTOR2：二号电机）
direction旋转方向 （STOP：停止）（FORWARD：往前） （BACKWARD：往后）
frequency频率
返回值无
